# 🏋️ Gym Management System

A full-stack web application for managing gym memberships, trainers, and assignments. 

### 🚀 Features
* **Multi-Role Access:** Separate portals for Admins, Trainers, and Members.
* **Secure Backend:** Powered by Spring Security with Basic Auth.
* **Dynamic UI:** Responsive Bootstrap 5 frontend with real-time toast notifications.
* **RESTful API:** Clean separation of concerns between Java and JavaScript.

### 🛠️ Tech Stack
* **Backend:** Java, Spring Boot, Spring Security, JPA/Hibernate.
* **Frontend:** JavaScript (ES6+), HTML5, CSS3, Bootstrap 5.
* **Database:**  MySQL.

### ⚙️ Setup
1. Run the Spring Boot application (backend).
2. Open `index.html` in your browser.
3. Login with `admin` / `admin123`.
